# SinRural — Redesign

Site institucional novo do **Sindicato dos Produtores Rurais de Passos** (SinRural).

> Substitui o site atual em `http://www.sinrural.com.br` (HTTP puro, CMS custom, URLs com IDs numéricos, sem mobile-first, sem área do associado).

## Stack

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **Tailwind CSS v4**
- **shadcn/ui** + Base UI
- **TypeScript**
- Deploy: **Vercel**

## Rodando localmente

```bash
npm install
npm run dev
```

Abre em http://localhost:3000.

## Build

```bash
npm run build
npm run start
```
